# Git操作总结

# 1.基础操作篇

  • git init 建立本地仓库

  • git remote add origin xxxxxxxxxxx 关联远程仓库

    • origin 为远程仓库名字可以有多个远程仓库

    • git remote -v 查看本地远程库
      1
    • bash vscode
      git remote add origin https://github.com/kczssd/Git-.git

  • git clone xxxxxxxxxx 克隆远程仓库

  • git add 把工作区文件提交到暂存区

    • bash vscode
      git add --all(当前项目里) | git add .(当前项目) | git add xxx(指定文件) 
    • git status 查看文件改变状态
      1

  • git commit -m"写对commit的描述" 提交到本地仓库

    • bash vscode
      git commit -m"add:a b c"

  • git push 推送到远程仓库

    • 第一次使用git push -u origin master

    • bash vscode
      git push -u origin master
      • master 可以切换push的远程仓库

    • git branch -a 查看本地和远程分支

# 2.代码撤销修改

未暂存时

  • git diff 列出所有的修改

    • git diff |git diff 具体文件
      -表示移除代码+表示增加代码

  • git checkout 撤销修改 | git clean -f/-df 撤销新增文件/文件夹

    • git checkout | git checkout 具体文件
      git clean -f | fit clean -df

未提交

  • git diff --cached 显示暂存区和本地仓库
    • git diff --cached |
  • git reset 暂存区到工作区
    • git reset --hard 回到工作区未修改状态

未推送

  • git diff master origin/master 比较本地仓库和远程仓库的差异
  • git diff branch1 branch2 比较两个分支的差异
  • git reset --hard origin/master 回退到与本地远程仓库一致
  • git reset --hard HEAD^ 回退到本地仓库上一个版本(版本回退)
    • git reset --hard 哈希值回退到指定版本

# 3.分支相关

  • git branch 查看本地分支
    • git branch -a 查看本地远程分支
  • git branch 分支名 创建分支
  • git checkout 分支名 切换分支
    • git checkout -b 分支名 创建并切换分支
  • git merge 分支名 把分支合并到本地分支

# 版本相关

  • git log 查看commit版本
    • git log --graph --decorate --abbrev-commit --all 图形化展示
  • git reflog 查看最近执行操作及所处的版本
